Deploy on GitHub Pages

  1. Create a public GitHub repo.
  2. Commit index.html and app.js.
  3. Enable Pages: branch main, folder / (root).
  4. Open the Pages URL and test by loading your CSV files.

Reusable: To adapt for other split datasets, change the schema and charts in app.js where marked.
